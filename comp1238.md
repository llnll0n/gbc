- one byte equals eight bits
- a file is just a stream of bytes

## Week 3 - Text Structures in Computing
- The concept of markup language comes from traditional publishing
- Markdown is easily convertible into HTML
- GitHub includes a built-in Markdown editor
## Week 4 - Utilizing text editors
- Keys like Control, Shift, and Alt are called modifier keys
- Touch typing is the ability to type without looking at the keys
- Wildcards are simple pattern-matching characters, commonly used when searching for files
## Week 5 - Command Line Interfaces
- The general structure is command [options] [arguments].
- SSH provides an encrypted connection between your computer and a remote server.
- scp: Secure Copy. Uses SSH to securely copy files between computers.
